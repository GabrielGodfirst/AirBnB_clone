# AirBnB Clone - Web Static

## Overview
This CLI project, "0x01. AirBnB clone - Web static," is part of a series
focused on creating a full-stack AirBnB clone. This specific project centers
on developing the static HTML and CSS components of the application.

## Project Structure
The project structure is organized as follows:
- **web_static**: Contains all static HTML and CSS files for the AirBnB clone.
    - **0-index.html**: Main HTML file, serving as the entry point.
    - **1-index.html**, **2-index.html**, ...: Additional HTML files for various
      sections/pages.
    - **styles**: CSS files for styling the HTML pages.
    - **images**: Image assets used in the application.

## Getting Started
To view the static pages locally, open the HTML files in a web browser.
Navigate through the HTML files by clicking links or opening desired files
directly.
